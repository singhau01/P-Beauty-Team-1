{{ block title }} p-beauty contest –– 說明 {{ endblock }} {{ block content }}

歡迎，感謝您參與今日的實驗！

待會您將與另外 {{ num_player_1 }} 位玩家，玩「 2⁄3 倍的平均數」的遊戲。

本遊戲共有 {{ C.NUM_ROUNDS }} 回合。

每一回合中，所有玩家皆需要從最小為 0，最大為 100 的區間裡，挑選一個整數。
待所有玩家輸入完畢後，系統將公布「 2⁄3 」倍，意即所有玩家選擇整數的平均數的 2⁄3 倍。

若您挑選的整數，在所有玩家中最接近「 2⁄3 倍的平均數」，您就是該回合的贏家，將可獲得報酬 {{ C.winning_prize }} 元新台幣。
若不是，可獲得報酬 {{ C.consolation_prize}} 元新台幣。
惟須注意，當最接近「 2⁄3 倍的平均數」的玩家有兩位以上時，該回合的所有贏家將均分報酬 {{ C.winning_prize }} 元新台幣。
{{ C.NUM_ROUNDS }} 回合結束後，將於結算頁面顯示您參與本次遊戲獲得的報酬總額（不含車馬費 100 元新台幣）。

例如：
以下是您與其他 2 名玩家在某一回合中選擇的數字。
您選擇的數字：56
其他 2 名玩家選擇的數字：7 99
在這個回合當中，平均數為 54，2⁄3 倍的平均數等於 36 。
由於 56 最接近獲勝數字 36，您是本回合的贏家，可獲得報酬 {{ C.winning_prize }} 元新台幣。
其他選擇 7 與 99 的玩家，可獲得報酬 {{ C.consolation_prize }} 元新台幣。

每一回合的決策時間是 {{ C.timeout_sec }} 秒。
倒數 {{ C.timer_sec }} 秒時，畫面會顯示倒數計時器。
倒數 {{ C.alert_sec }} 秒時，畫面會出現「請您儘速作答」字樣提醒您作答。
若您未在限制時間內做出決策，該回合僅能獲得報酬 {{ C.noplaying_prize }} 元新台幣。
